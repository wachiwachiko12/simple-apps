# 資格試験 問題ランダム出題ツール

- 英語名: Quiz Tool / Random Question Generator
- フォルダ: apps/quiz-tool/
- キーワード: 資格試験 問題集 無料, ITパスポート 問題 ランダム, 試験対策 ツール
- 推定月間検索数: 5,000〜15,000
- ステータス: live

## 機能一覧

1. CSV / JSON ファイルのドラッグ&ドロップ or クリックインポート
2. サンプルCSVダウンロード（ITパスポート風5問）
3. 出題モード: ランダム / 苦手優先 / 全問順番
4. 4択クイズUI — 選択肢ボタン→正誤表示→解説→次の問題
5. 進捗バー・正解率・残り問題数のリアルタイム表示
6. 苦手問題フラグ（間違えた問題をlocalStorageに保存）
7. セッション完了画面 — 総合スコア・苦手問題一覧・再挑戦

## 広告配置計画

- `#ad-top`: ファーストビュー直下（インポートセクション上部）
- `#ad-result`: 結果表示直下（使い方セクション上部）
