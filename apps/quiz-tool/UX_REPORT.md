# UXレビュー — 資格試験 問題ランダム出題ツール

**総合判定: ✅ 承認**

ペルソナ: ITパスポート・FP・宅建を独学中の社会人（20〜40代）。スキマ時間にスマホで問題練習したい。Quizletは有料で使えないため無料代替ツールを探している。

---

## 1. ファーストビュー ✅

**評価: 良好**

- ヘッダーに「資格試験 問題ランダム出題ツール」「CSVで自分の問題集を持ち込んで本格学習。ITパスポート・FP・宅建・英検 対応。」と明記されており、3秒以内に用途が把握できる。
- ドロップゾーンが青い破線ボックスで視覚的に目立つ。「CSVまたはJSONファイルをここにドロップ」「ファイルを選択」「サンプルCSVをダウンロード」の3択が明確。
- ad-topが import-section より先に来ているが、.ad-unit に height/background 指定がなく未審査段階では非表示となるため、視覚的な阻害はない。

---

## 2. 入力UX ✅

**評価: 良好**

- ドラッグ&ドロップ（dragover/dragleave/drop イベント）とクリック（`label[for=file-input]`）の両方に対応。
- dropZone 全体クリックでも `fileInput.click()` が発火し、`file-btn` label クリック時の二重発火は `e.target.classList.contains('file-btn')` チェックで防止済み。
- サンプルCSV（修正後）にITパスポート・FP・宅建の例題をそれぞれ追加。ユーザーが自分の試験ジャンルを確認しながら書式を学べる。
- ファイル読み込み後に「ファイル名 (N問)」がグリーン背景で表示され、読み込み成功が一目でわかる。

---

## 3. クイズUX ✅

**評価: 良好**

- 選択肢ボタン min-height: 56px で44px基準を超過 ✅
- A/B/C/D のキーが円形バッジで視覚的に識別しやすい。
- 正解時: 緑背景 + 「正解！」(正解メッセージ緑色) + 選択肢ボタンが緑枠で強調。
- 不正解時: 赤背景 + 「不正解 — 正解は X」(赤色) + 誤答ボタン赤枠、正答ボタン緑枠で両方明示。
- 解説テキストがフィードバック欄に即表示される。
- プログレスバーで残り問題数が可視化されている。
- 最後の問題では「次の問題 →」が「結果を見る」に変わる。

---

## 4. エラー処理 ✅（修正実施）

**評価: 修正後・良好**

| ケース | 修正前 | 修正後 |
|--------|--------|--------|
| 問題0件のCSV | alert「問題が1問も読み込めませんでした」 ✅ | 変更なし |
| JSON が配列でない | throw → alert「ファイルの読み込みに失敗しました」 ✅ | 変更なし |
| 正解列が A〜D 以外 | 無言でスルーされ、その問題で正解不能 ❌ | スキップ + alert で該当行番号を通知 ✅ |
| 正解列が A〜D 以外（JSON）| フィルタなく混入 ❌ | `VALID_ANSWERS` フィルタ追加 ✅ |

- **修正内容**: `parseCSV` に `VALID_ANSWERS` セットを導入し、A〜D 以外の正解列を持つ行をスキップしてユーザーに通知するロジックを追加。`parseJSON` にも同様のフィルタを追加。

## 5. モバイル ✅

**評価: 良好**

- `<meta name="viewport" content="width=device-width, initial-scale=1.0, maximum-scale=1.0">` 設定済み（iOSピンチズーム防止）。
- choice-btn: min-height 56px ✅
- .btn（学習開始・次の問題等）: min-height 44px ✅
- .file-btn: min-height 44px / line-height 44px ✅
- .back-link: min-height 44px / line-height 44px ✅
- .mode-row select: min-height 44px ✅
- 420px以下では choices-grid が1列になり、縦長テキストでも読みやすい。
- `-webkit-tap-highlight-color: transparent` 設定済み（タップ時のグレーフラッシュ防止）。

## 7. コンテンツ ✅（修正実施）

**評価: 修正後・良好**

- 「使い方」4ステップが番号付きリストで簡潔にまとめられている ✅
- CSVフォーマット説明のコードブロックが見やすい（ダークテーマ）✅
- **修正内容**: CSVフォーマット例にITパスポート・FP・宅建の3ジャンルの例を追加。「正解列には A・B・C・D のいずれかを記入してください（大文字）」と明記。サンプルCSVが多ジャンル対応であることも注記。
- サンプルCSVも7問（ITパスポート3問・FP2問・宅建2問）に拡充し、ダウンロード後すぐに各ジャンルの書式を確認できる。
- 「このツールについて」にQuizlet代替である旨・localStorageで苦手保存される旨が記載されており信頼性・SEO的にも充実。

## 8. 広告 ✅

**評価: 良好**

- `.ad-unit` の CSS は `margin: 1.25rem 0;` のみ。
- `min-height` や `background` 等の不正なスタイルなし ✅
- `aria-hidden="true"` が設定されスクリーンリーダーに影響しない ✅
