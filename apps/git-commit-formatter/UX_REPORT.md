## 残課題（あれば）

- h1テキスト「Gitコミットメッセージフォーマッター — Conventional Commits準拠」がモバイルで3行折り返しになる場合がある。短縮版のaria-labelやvisually-hiddenを追加するかsubtitleに詳細を移す構成も検討できる（ブロッカーではない）
- 件名入力のカーソルが50文字を超えると警告色になるが、視覚的フィードバックのみで音声読み上げへの通知がない。`aria-live` または `role="status"` での文字数アナウンスを将来的に検討
