Markdownテーブル作成ツール — スマホ対応・無料・GUIで直感操作

ExcelライクなGUI操作でMarkdownテーブルを無料作成。スマホ・PC対応。TableFlip代替として登録不要ですぐ使えます。

使い方

      
    
        
  1. セルをクリックして直接テキストを入力します。
    2. 
        
  2. Tab キーで右のセルへ、Shift+Tab で左のセルへ移動します。
    3. 
        
  3. Enter キーで下のセルへ、矢印キー（↑↓←→）でも移動できます。
    4. 
        
  4. 「行を追加 / 削除」「列を追加 / 削除」ボタンでテーブルの大きさを調整します。
    5. 
        
  5. 各列の下にある 揃え ドロップダウンで、左揃え・中央・右揃えを設定します。
    6. 
        
  6. 「Markdownをコピー」ボタンをクリックすると、生成されたMarkdownがクリップボードに保存されます。
    7. 
        
  7. 既存のMarkdownテーブルを「読み込む」フォームに貼り付けて「テーブルに読み込む」をクリックすると、GUIで編集できます。
    8. 
        
  8. 入力内容はブラウザの localStorage に自動保存されるため、ページを閉じても次回開いたときに続きから編集できます。
    9. 
      


      
このツールについて

      

        Markdownテーブル作成ツールは、GitHubのREADMEやQiita・Zennの記事、Notionドキュメントなどで使われるMarkdown形式のテーブルを、Excelライクな直感的GUIで作成・編集できる完全無料のブラウザツールです。従来はテキストエディタで | と - を手打ちする必要がありましたが、このツールならセルをクリックして文字を入力するだけでMarkdownコードが自動生成されます。Mac専用有料アプリ「TableFlip」の完全無料代替として、OS・ブラウザを問わず利用できます。生成したMarkdownはワンクリックでコピーでき、開発ドキュメントや技術ブログの執筆効率を大幅に向上させます。アカウント登録不要、サーバーへのデータ送信なし、データはすべてお使いのブラウザ内に保存されます。
