## 機能概要
1. CSS Gridモード — 列数・行数・gapをGUIで設定、セルクリックでspan設定、プリセット5種
2. Flexboxモード — direction / wrap / justify / align などを設定、リアルタイムプレビュー
3. CSS / Tailwind CSS 出力タブ切り替え + 1クリックコピー
4. localStorage 自動保存
5. プリセット: 2カラム / 3カラム / ヘッダー+サイドバー+メイン / カード格子 / Holy Grail
