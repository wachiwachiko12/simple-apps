ふりがな自動付与ツールの使い方

  1. 上のテキストエリアにふりがなを付けたい日本語テキストを貼り付けるか入力します（最大2000文字）。
  2. 「ふりがなを付ける漢字レベル」で対象とする漢字の難易度を選択します。例えば「小学3年生以上」を選ぶと、3年生で習う漢字以上にのみルビが付きます。
  3. 「出力形式」で希望する形式を選んでください。HTMLルビタグ形式はWebページ埋め込みに、かっこ形式はメール・文書作成に最適です。
  4. 「ふりがなを付ける」ボタンを押すと変換が実行されます。
  5. プレビューエリアで実際のルビ表示を確認し、「コピー」ボタンで結果をクリップボードにコピーします。

ふりがな（ルビ）とは

ふりがなとは、漢字の読み方を示すために小さな文字（ひらがな・カタカナ）で添えた注記のことです。日本では、子ども向けの書籍・教材・絵本や、外国人向けの日本語テキスト、公的文書など幅広く使われています。Webページでは HTML の <ruby> タグを使って実装でき、漢字の上（縦書き時は右）にルビが表示されます。本ツールは形態素解析ライブラリ kuromoji.js を使用して漢字の読みを自動取得するため、手作業なしで大量のテキストにまとめてふりがなを付けることができます。教材作成・子ども向けコンテンツ制作・SNS投稿など、さまざまな場面でご活用ください。

出力形式の説明

形式 例 用途
HTMLルビタグ <ruby>漢字<rt>かんじ</rt></ruby> Webページ・HTML文書への埋め込み
かっこ形式 漢字（かんじ） メール・Word文書・SNS投稿
テキスト確認用 かんじ 読み仮名だけを確認したいとき

学年別漢字レベルについて

文部科学省の学習指導要領に基づく配当漢字を参考に、指定した学年以上で習う漢字のみにルビを付ける機能です。例えば「小学3年生以上」を選ぶと、1・2年生の漢字（山・川・学など）にはルビが付かず、3年生以降で習う漢字にのみルビが付きます。子ども向け読み物の制作や日本語学習教材の作成に便利です。
