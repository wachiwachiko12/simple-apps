# UXレビューレポート — Webパフォーマンス予算ツール

**レビュー日**: 2026-05-29
**総合判定**: ✅承認

## 修正済み項目

- 広告ユニット3か所（`#ad-top` / `#ad-result` / `#ad-bottom`）のインラインスタイル `style="margin: 1.25rem 0;"` を削除。CSS `.ad-unit` で同値が既に定義されており重複していた。`min-height` / `background-color` は含まれておらず、AdSenseポリシー上の問題はなかった。

## 評価

| 軸 | 評価 | コメント |
|---|---|---|
| ファーストビュー | ★★★★☆ | h1とsubtitleで「バンドルサイズ管理・Core Web Vitals目標設定」が3秒以内に伝わる。プリセットボタンが目立ち、初動の迷いが少ない。ただしターゲット（フロントエンドエンジニア）以外には専門用語が多く、入門者への補足説明がファーストビュー内にない |
| 入力UX | ★★★★★ | 全入力要素に `aria-label` と `placeholder` が付与されている。モバイル時はラベル「予算上限:」「実際のサイズ:」が疑似要素で表示され、デスクトップではテーブルヘッダーに切り替わる二段構えの設計が丁寧。`min-height: 44px` をすべての入力・ボタンが満たしている |
| 結果表示 | ★★★★★ | プログレスバーがリアルタイムで色変化（緑→黄→赤）し、使用率%が数値でも表示される。CWVカードはスライダー操作即時でGood/Needs Improvement/Poorがバッジ+カード枠色で視覚的に判断できる。凡例（良好・注意・超過）が表の下に配置されており意味の確認が容易 |
| エラー処理 | ★★★☆☆ | 未入力・0入力時はゲージが「—」表示になり誤表示を防いでいる。負数入力は `min="0"` で抑制済み。ただし予算入力なしで実績だけ入力した場合はゲージが「—」のままで、ユーザーが「なぜ動かないか」気づきにくい。また `confirm()` でリセット確認ダイアログを出しているが、ダイアログのUIはブラウザデフォルトで統一感がない |
| モバイル | ★★★★☆ | `maximum-scale=1.0` 設定済みでiOSの自動ズームを防止。モバイル時にリソーステーブルが縦積みレイアウトに切り替わり可読性を維持。CWVスライダーのサムが22pxで操作しやすい。プリセットが2列グリッドで表示されタップしやすい。560px以上でCWVカードが3列になるが、モバイルでは1列表示になっており問題なし |
| 信頼性 | ★★★★★ | フッターに `/privacy-policy.html` リンクあり。「プライバシーについて」セクションでデータのブラウザ内完結処理を明記。プリセット値の根拠（HTTP Archive・Web Almanac）を説明するセクションが設けられており、数値の信頼性が担保されている |
| コンテンツ | ★★★★★ | h1に「Webパフォーマンス予算ツール」「バンドルサイズ」「Core Web Vitals」が含まれ、titleタグ・meta descriptionとキーワードが一致。「パフォーマンス予算とは」「Core Web Vitals（CWV）について」「プリセット値の根拠」の3セクションがSEOコンテンツとして機能し、対象キーワードの網羅性が高い |
| 広告配置 | ★★★★★ | `.ad-unit` はCSSで `margin: 1.25rem 0;` のみ定義。`min-height` / `background-color` なし。インラインスタイルの重複は修正済み。広告ユニットは3か所（ヘッダー直下・CWVセクション下・コンテンツ末尾）でコンテンツとのバランスが取れている。`aria-hidden="true"` でスクリーンリーダーに広告枠を無視させている |

## 残課題（あれば）

- **エラーヒント**: 実績値を入力したが予算値が未入力の場合、「予算上限を入力するとゲージが表示されます」などのインラインヒントがあると初回ユーザーの迷いが減る（優先度: 低）
- **リセット確認UI**: `confirm()` ダイアログはブラウザごとにスタイルが異なり、デザインとの一貫性が崩れる。将来的にカスタムモーダルへの置き換えを検討（優先度: 低）
- **入門者向け導線**: ターゲットがフロントエンドエンジニアのため現時点では問題ないが、「Chrome DevToolsでのサイズ計測方法」へのリンクを使い方セクションに加えると実用性が向上する（優先度: 低）
