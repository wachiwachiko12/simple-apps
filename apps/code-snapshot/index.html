コードスニペット画像化ツール

コードをPNG画像に変換 — Zenn・Qiita・SNS投稿に最適

プレビュー

使い方

  1. 下のテキストエリアにコードを貼り付けます。
  2. 言語・テーマ・背景色・フォントサイズを設定します。
  3. 右側のプレビューでリアルタイムに確認します。
  4. 「PNG画像でダウンロード」ボタンを押して画像を保存します。
  5. 保存した画像をZenn・Qiita・X（Twitter）に投稿します。

コードスニペット画像化ツールとは

コードスニペット画像化ツールは、プログラムのソースコードを美しいPNG画像に変換できるWebサービスです。 エンジニアや技術ブロガーがZenn・Qiita・X（旧Twitter）・note などのSNSにコードを投稿する際、 コードブロックが使えないプラットフォームでも画像として共有できるようになります。 Carbon（carbon.now.sh）の日本語フォント対応版として、日本語コメントや日本語変数名も正しく表示されます。 シンタックスハイライトにはHighlight.jsを使用し、JavaScriptやPython・Go・Rustなど12言語に対応。 macOS風のウィンドウボタン（赤・黄・緑のドット）でリアルなエディタ風の見た目を演出できます。 登録不要・完全無料でブラウザだけで動作します。

対応テーマ一覧

  • Dark（デフォルト） — 定番のダークテーマ。目に優しく視認性が高い。
  • Light — 明るい背景のライトテーマ。印刷にも適している。
  • Dracula — 人気のDraculaカラーパレット。紫・ピンク系の配色。
  • One Dark — Atom/VS Code ユーザーにおなじみの落ち着いた暗色テーマ。
  • Solarized Dark — 目に優しい低コントラストの定番テーマ。
