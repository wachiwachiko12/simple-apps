# タイムライン作成ツール (Timeline Visualizer)

## アプリ情報

- **日本語名**: タイムライン作成ツール
- **英語名**: Timeline Visualizer
- **URL**: https://keisanlab.jp/apps/timeline-visualizer/
- **ステータス**: live

## ターゲットキーワード

- タイムライン 作り方
- タイムライン 作成 ツール 無料
- 年表 作成 オンライン
- 時系列 可視化

## 推定月間検索数

- 「タイムライン 作り方」: 約 8,000
- 「タイムライン 作成 ツール 無料」: 約 2,000
- 「年表 作成 オンライン」: 約 1,500

## 機能

- イベント（タイトル・日付・カテゴリ・カラー・説明）の追加・編集・削除
- 水平 / 垂直レイアウト切り替え
- PNG画像エクスポート（html2canvas）
- JSON エクスポート / インポート
- ローカルストレージ自動保存
- サンプルデータ読み込み
