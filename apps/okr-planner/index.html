サマリー

使い方

  1. 画面上部のタブ（Q1〜Q4）から管理したい四半期を選択します。
  2. 「+ Objectiveを追加」ボタンで目標（Objective）を最大4つ作成します。
  3. 各ObjectiveカードのKey Result（成果指標）欄に、目標値・現在値・単位を入力します。
  4. 達成率が自動計算され、カラーバーとサマリーに反映されます。
  5. データはブラウザのlocalStorageに自動保存されます。ページを閉じても保持されます。

OKRとは

OKR（Objectives and Key Results）は、Google・Intelなどが採用する目標管理フレームワークです。「Objective（何を達成するか）」と「Key Result（達成をどう測定するか）」をセットで設定することで、曖昧な目標を定量化し、進捗を可視化できます。個人・フリーランスが四半期単位で使うことで、売上目標・スキルアップ・案件獲得などの優先事項を整理し、集中して行動できる効果があります。従来はResily・Wistantなどの有料ツールが主流でしたが、このツールは登録不要・完全無料でお使いいただけます。

進捗バーの見方

  • 赤（0〜40%）: 未達・要注意。アクションの見直しが必要です。
  • 黄（41〜70%）: 進行中。このまま継続するか戦略を調整しましょう。
  • 緑（71〜100%）: 順調。目標達成に向けて良いペースです。

フリーランス向けOKR設定例

  • Objective: 収入目標を達成する — KR: 月次売上100万円 / 新規クライアント3件獲得
  • Objective: スキルアップを実現する — KR: 資格1つ取得 / 技術記事10本執筆
  • Objective: 案件獲得率を上げる — KR: 提案数20件 / 成約率30%達成
