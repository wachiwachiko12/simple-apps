# タイピング練習ツール

- 英語名: Typing Practice
- フォルダ: apps/typing-practice/
- キーワード: タイピング練習 無料, WPM 計測, ローマ字 タイピング, e-typing 代替
- 推定月間検索数: 10,000〜30,000
- ステータス: live

## 機能概要

- 3モード: 英語 / ローマ字（日本語） / かな入力
- テキストカテゴリ: 英語基本単語100選 / プログラミング用語 / 英文名言 / ことわざ / 技術用語 / 短文練習
- リアルタイム計測: WPM・正確率・経過時間・文字ハイライト（正誤色分け）
- セッション完了画面: WPM・正確率・ミス数・ランク（S/A/B/C/D）
- ベストスコア: localStorageでモード×カテゴリ別に最高WPMを保存

## 広告配置計画

- ad-top: ツール上部（ファーストビュー内）
- ad-result: 使い方セクション直前

## 技術メモ

- ローマ字変換テーブル: 最長一致（4文字→1文字）で変換
- かなモード: IME入力対応（環境依存）
- スコア管理: localStorage キー = `typing-best-{mode}-{category}`
