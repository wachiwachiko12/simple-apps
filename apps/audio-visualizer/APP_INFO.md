## ターゲット情報

- **ターゲットユーザー**: ポッドキャスター・音楽制作者・SNS投稿者
- **ターゲットキーワード**: 音声波形ビジュアライザー、波形 画像 作成、ポッドキャスト サムネイル、SNS 音楽 画像
- **推定月間検索数**: 500〜1,500（ニッチ・競合少）
- **差別化ポイント**: 完全ローカル処理・透かしなし・無料・登録不要

## 主要機能

1. 音声ファイル読み込み（MP3/WAV/OGG、ドラッグ&ドロップ）
2. 波形描画（バー型・ライン型）— Web Audio API + Canvas API
3. カラーカスタマイズ（背景色・グラデーション波形色・テキスト色）
4. サイズプリセット（1200×630 OGP / 1080×1080 Instagram / 1500×500 Twitter）
5. テキストオーバーレイ（曲名・アーティスト名）
6. PNG保存（透かしなし）
7. マイク入力リアルタイム波形

| #ad-bottom | コンテンツ下部 | 低〜中 |

## 技術仕様

- Web Audio API（AudioContext.decodeAudioData / AnalyserNode）
- Canvas API（2D Context、グラデーション、toBlob PNG出力）
- getUserMedia（マイク入力）
- 外部ライブラリなし・サーバー通信なし
