# 職務経歴書ジェネレーター

- アプリ名（日本語）: 職務経歴書ジェネレーター
- アプリ名（英語）: Resume Generator
- ターゲットキーワード: 職務経歴書 テンプレート 無料, 職務経歴書 作成 ツール
- 推定月間検索数: 職務経歴書 テンプレート 無料（約 14,000〜22,000 / 月）
- 広告配置計画: howto-section 内に A8.net 転職・人材系アフィリエイトリンクを設置
- ステータス: live

## 実装概要

- 左ペイン（入力フォーム）＋右ペイン（リアルタイムプレビュー）の2カラムレイアウト
- モバイルは「入力」「プレビュー」タブで切り替え
- タブ構成: 基本情報 / 職務要約 / 職務経歴 / スキル・資格 / 希望条件
- localStorage 自動保存（外部サーバー送信なし）
- 「印刷 / PDF保存」ボタンで window.print() → ブラウザ印刷機能でPDF出力
- スキルはタグ形式（Enter キーで追加・×で削除）
- 職歴は複数追加可能・プレビューは新しい順に表示

## 技術スタック

- HTML5 / CSS3 / Vanilla JS のみ（CDN不使用・外部API不使用）
- localStorage によるデータ永続化
- @media print でフォームを非表示・プレビューのみ印刷

## ファイル構成

- index.html — メインページ（SEO対応・GA4・OGP・アフィリエイト含む）
- style.css — モバイルファースト・レスポンシブ・印刷対応
- script.js — 全ロジック（localStorage・プレビュー更新・タグ入力・モバイル切り替え）
- APP_INFO.md — このファイル
