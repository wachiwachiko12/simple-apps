## 機能一覧

1. ベースフォントサイズ スライダー設定（12〜24px、px/rem切替）
2. 8種スケール比率プリセット（Minor Second〜Golden Ratio）
3. 行間（line-height）スライダー
4. 字間（letter-spacing）スライダー
5. 8ステップ（h1〜h6 + p + small）フォントサイズ自動計算
6. 日本語サンプルテキストによるリアルタイムプレビュー
7. CSS変数出力（`:root { --text-5xl: Xrem; ... }`）
8. Tailwind CSS `fontSize` 設定出力
9. ワンクリックコピー（CSS/Tailwind タブ切替）
10. localStorage 自動保存
