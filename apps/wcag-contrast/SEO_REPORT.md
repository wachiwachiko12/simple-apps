## 差別化戦略

### 勝てるポイント

1. **WCAG 2.2 完全対応を前面に出す**: 競合の多くがWCAG 2.1止まり。2.2の新基準（達成基準1.4.11 非テキストのコントラスト等）を明示することで専門性を示せる。
2. **色覚シミュレーション統合**: 1型・2型・3型色覚（P型・D型・T型）の見え方プレビューを内蔵しているツールは日本語圏に少ない。これは大きな差別化要素。
3. **スマホ完結UIの完成度**: 既存ツールはPCデザイナー向けが多く、スマホでのカラーピッカー操作が壊れているケースが多い。モバイルファーストで優位に立てる。
4. **JIS X 8341-3:2016 対応の言及**: 日本の官公庁・自治体サイト担当者向けに国内規格名も明記することで検索流入を広げられる。
5. **AA/AAA 判定 + 合格色の自動提案**: 単なるチェックではなく「修正案提示」まで行うことでツールの完結性が高まる。

### 実装すべき機能

- カラーピッカー（HEX / RGB / HSL 入力対応）
- リアルタイムコントラスト比計算
- WCAG 2.2 AA / AAA 合否表示（通常テキスト・大テキスト・UI部品の3区分）
- 合格する近似色の自動提案
- テキスト・背景色スワップボタン
- 色覚シミュレーション（P型・D型プレビュー）
- 結果のSNSシェア / クリップボードコピー
- WCAG基準の日本語解説セクション（コンテンツSEO兼用）

## 総評・勝算

**競合強度: 中程度 / 勝算: 高め**

日本語圏のWCAGコントラストチェッカー市場は、複数のツールが存在するものの、WCAG 2.2完全対応・色覚シミュレーション・モバイル最適化を全て備えたものは少ない。特に「デザイナーがスマホで手軽に確認できる」というニーズは未充足に近く、差別化が十分可能な領域。

2024〜2026年にかけて日本でもウェブアクセシビリティへの関心が高まっており（障害者差別解消法改正・官公庁サイトのJIS X 8341義務化等）、検索数は増加傾向にある。月間1万以上のセッション獲得が現実的に見込める。

コンテンツSEO戦略として「WCAG AA基準とは」「コントラスト比の計算式」「色覚バリアフリーデザイン入門」などの解説セクションをページ内に設けることで、情報収集型（Know型）の流入も取り込める。
