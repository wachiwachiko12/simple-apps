# フリーランス確定申告シミュレーター

- **デプロイ日**: 2026-05-10
- **URL**: /apps/freelance-tax-simulator/
- **カテゴリ**: 金融・税務
- **広告RPM**: $12（金融ニッチ最高水準）
- **対象ユーザー**: フリーランス・個人事業主

## 機能

- 年間売上・経費から事業所得を計算
- 青色申告特別控除（65万・55万・10万・白色）対応
- 所得税（累進課税・復興特別所得税2.1%含む）計算
- 住民税（所得割10% + 均等割5,000円）計算
- 消費税（免税事業者・簡易課税・本則課税）計算
- 合計納税額・手取り概算を表示
