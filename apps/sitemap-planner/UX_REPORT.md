# UX Review Report — 無料サイトマップ作成ツール

**レビュー日**: 2026-05-26
**ペルソナ**: Webディレクター（28歳）。Octopus.doの英語UIに不満を感じており、「登録不要でさっとサイト構成図を作れる日本語ツール」を探してたどり着いた初回訪問者。

---

## 8軸評価

### 1. ファーストビュー ✅
ヘッダーに「無料サイトマップ作成ツール」というh1と「Webサイト構成図をツリー図でビジュアル設計 — 登録不要・完全無料」というサブタイトルが即座に目に入る。ページを開いた3秒以内に目的を把握できる。サンプルツリーが初期表示されており、「こういうものが作れるのか」と直感的に理解できる。

### 2. 入力UX ✅
- ページ名入力欄に `placeholder="例: サービス概要"` が設定されており入力例が明確。
- `<label>` タグがすべての入力要素に紐付いており、スクリーンリーダー対応。
- 「追加先のノードを選択してから追加ボタンを押してください」というヒントテキストが操作手順を案内している。
- Enterキーでも追加実行できる（keyboard shortcut対応）。

### 3. 結果表示 ✅
- カラーコード付きバッジ（TOP/一覧/詳細/フォーム/特集/LP）で階層と種別が視覚的に区別できる。
- ノードにホバー時シャドウが増し、選択状態（青枠）も明確。
- ズームイン/アウトで大規模ツリーでも確認できる。
- PNG書き出し結果のファイル名に日付が含まれており管理しやすい。

### 4. エラー処理 ✅
- ページ名未入力で追加ボタン押下時 → トースト「ページ名を入力してください」
- ノード未選択で追加ボタン押下時 → トースト「追加先のノードを選択してください」
- ノード削除前に confirm ダイアログで誤操作防止
- リセット前にも confirm ダイアログ
- JSON インポート失敗時もトーストでフィードバック
- ルートノードは削除不可（「ルートノードは削除できません」）

### 5. モバイル ⚠️ → 修正済み
**発見した問題:**
- ズームボタン（`+`/`−`）が `width: 32px; height: 32px; min-height: 32px` と44px未満だった。

**適用した修正:**
- `.btn-zoom` を `width: 44px; height: 44px; min-height: 44px` に変更。

その他:
- viewport に `maximum-scale=1.0` 設定済み（iOSズーム防止）✅
- モバイルではパネルが上部に表示され、ツリーが下に続くレイアウト（縦スタック）✅
- `btn-primary` / `btn-secondary` / `btn-danger` はすべて `min-height: 44px` 以上✅
- 水平スクロールはツリーコンテナ内のみ発生（ページ全体ではなし）✅

### 6. 信頼性 ✅
- フッターに「プライバシーポリシー」へのリンク設置済み。
- 「使い方」セクション（7ステップの番号付きリスト）と「サイトマップ・IA設計ツールとは」説明文が充実。
- ローカルストレージ自動保存・Octopus.doとの差別化説明・JSON共有方法など、ユーザーの不安を取り除くコンテンツあり。

### 7. コンテンツSEO ⚠️ → 修正済み
**発見した問題:**
- `<title>` が `サイトマップ・IA設計ツール 無料 | Keisanlab` で、SEO_REPORT推奨の `サイトマップ作成ツール 無料 | IA設計・構成図をツリー図で可視化` と乖離。
- `<h1>` が `サイトマップ・IA設計ツール` で、推奨の `無料サイトマップ作成ツール — Webサイト構成図をツリー図でビジュアル設計` と乖離。
- `<meta name="description">` がSEO_REPORT推奨文と異なっていた。
- `og:title` も古いテキストだった。

**適用した修正:**
- `<title>` → `サイトマップ作成ツール 無料 | IA設計・構成図をツリー図で可視化`
- `<h1>` → `無料サイトマップ作成ツール`（副題で補足）
- `<meta name="description">` → SEO_REPORT推奨文に変更
- `<meta property="og:title">` → タイトルと統一

その他（問題なし）:
- `og:image` / `twitter:image` → `/images/ogp-sitemap-planner.png` を指定済み ✅
- `canonical` タグ設定済み ✅

### 8. 広告配置 ✅
`.ad-unit` のCSSは `margin: 1.25rem 0;` のみ。`min-height` も `background` も設定なし。AdSenseポリシー準拠。

主要機能（ツリー編集・ノード追加/削除/折りたたみ・PNG書き出し・JSON入出力・ローカルストレージ保存）は完全実装済み。ブロッカーはなし。修正5件を適用済みのため、このままデプロイ可能。
