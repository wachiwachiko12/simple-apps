# ポモドーロ×タスクリスト統合タイマー

- **英語名**: Pomodoro Timer with Task List
- **フォルダ**: apps/pomodoro/
- **キーワード**: ポモドーロ 無料, ポモドーロタイマー タスク管理, 集中タイマー 無料
- **推定月間検索数**: 10,000〜30,000
- **ステータス**: live

## 広告配置計画

| 位置 | ID | 期待クリック率 |
|------|-----|--------------|
| ツール上部 | ad-top | 中 |
| 進捗カウント下 | ad-result | 高（タイマー完了後に目線が移動） |

## ターゲットユーザー

- テレワーカー（自宅での集中力管理）
- 学生（勉強セッション管理）
- フリーランス（作業時間の記録・可視化）

## 競合分析メモ

- Focus Booster: $4.99/月 → 本アプリは完全無料の代替
