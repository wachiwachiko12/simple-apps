# UX レビューレポート — ポモドーロタイマー × タスクリスト

**レビュー日**: 2026-05-26  
**レビュアー**: ux-reviewer  
**ペルソナ**: テレワーク中の会社員（30代）。スマホとPCを両方使う。ポモドーロテクニックを聞いたことがあるが初めて使うWebツール。

---

## 8軸評価

### 1. ファーストビュー ✅
- ヘッダーの h1「ポモドーロタイマー × タスクリスト」とサブタイトル「25分集中 + 休憩サイクルで生産性アップ。無料・登録不要。」で目的が3秒以内に明確に伝わる。
- 赤〜オレンジのグラデーション背景と大きな円形タイマー表示（25:00）が視覚的にインパクトがあり、直感的にタイマーアプリと分かる。
- 「開始」ボタンが白く目立ち、CTAとして機能している。

### 2. 入力UX ✅
- タスク名入力欄にプレースホルダー「タスク名を入力...」があり、何を入力すべきか明確。
- 予定ポモドーロ数はセレクトボックスで選択でき、デフォルト3が適切。
- タスク追加ボタン「+ 追加」はオレンジ色で視覚的に強調されている。
- フォームはEnterキーで送信でき、入力後フォーカスが戻るためスムーズな連続追加が可能。
- 時間設定ボタン（15/20/25/30/45分）はタイマー中に変更できない制御が実装済みで誤操作を防いでいる。

### 3. 結果表示 ✅
- 円形プログレスバーで残り時間が視覚的に把握できる。
- モードバッジ（「集中タイム」「短い休憩」「長い休憩」）の色変化（赤→緑→深緑）でモード切替が一目でわかる。
- 今日のポモドーロ数・完了タスク数が常時表示されており達成感を高める設計。
- タスク一覧でトマトアイコン「進捗/予定」の表示が直感的。

### 4. エラー処理 ✅
- 空文字のタスク追加は`if (!name) return`で防止済み。
- localStorage の読み書きは try/catch でラップされ、クォータ超過などの例外をサイレント処理。
- Web Audio API が非対応の環境でも try/catch で無音フォールバック。
- タスク名は80文字のmaxlength制限あり。
- XSSエスケープ関数（escapeHtml / escapeAttr）が実装されており、動的生成HTMLも安全。
- アクティブタスクが完了状態になったとき自動解除される処理あり。

### 5. モバイル ✅
- viewport に `maximum-scale=1.0` が設定されておりiOSのフォームズームを防止。
- ほぼすべての操作ボタン（タイマー制御・タスク操作・時間設定・タスク追加）に `min-height: 44px` が設定されている。
- 主要アクション「開始」ボタン（`.btn-start`）は修正により `min-height: 56px` に拡大済み。
- `max-width: 720px` のセンタリングレイアウトで横スクロールは発生しない。
- 360px以下の超小画面向けのメディアクエリも実装済み。

### 6. 信頼性 ✅
- フッターにプライバシーポリシーへのリンクあり。
- 「使い方」セクション（番号付き手順・ポモドーロテクニックの解説・FAQ）が充実しており、初めてのユーザーを丁寧にガイドしている。
- 「登録不要・完全無料」の明示がある。
- タスクデータがブラウザに保存されることをFAQで説明しており、ユーザーの不安を解消している。

### 7. コンテンツSEO ✅
- h1「ポモドーロタイマー × タスクリスト」はSEO_REPORT.mdの推奨と一致。
- title タグを「ポモドーロタイマー × タスクリスト | 無料・登録不要 — keisanlab」に更新（SEO_REPORT.md の推奨に準拠）。
- meta description を「ポモドーロタイマーとタスクリストを1画面で管理。25分集中＋5分休憩サイクルを自動化し、仕事・勉強・テレワークの集中力を上げる無料Webツール。登録・インストール不要。」（約100文字）に更新。「インストール不要」「テレワーク」「仕事・勉強」を追加しクリック率向上を図った。
- og:image / twitter:image に `/images/ogp-pomodoro.png` が設定済み。
- canonical URL 設定済み。
- GA4 タグ（G-DK2WQM49L6）が `<meta charset>` 直後に配置済み。

### 8. 広告配置 ✅
- `.ad-unit` の CSS は `margin: 1.25rem 0;` のみ。`min-height` も `background` も存在しない。規定に完全準拠。
- 広告配置箇所（ツール上部・進捗カウント下）はSEO_REPORT.mdの推奨通り。
